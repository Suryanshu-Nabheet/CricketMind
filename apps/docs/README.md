# CricketMind - Technical Documentation Site

This directory contains the documentation-scoped static website for **CricketMind**, presenting development setup guidelines, system architectures, and API credential workflows.

---

## Development Setup

1. **Launch Docs Server:**
   From the repository root, run:
   ```bash
   pnpm --filter=docs dev
   ```

2. **Access Docs Portal:**
   Open [http://localhost:3001](http://localhost:3001).
